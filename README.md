\# Customer Churn Prediction Using Machine Learning



\## Project Overview

This project predicts customer churn using machine learning classification algorithms.



\## Models Used

\- Logistic Regression

\- Decision Tree

\- Random Forest



\## Best Model

Random Forest achieved the best performance:

\- Accuracy: 100%

\- ROC-AUC: 1.00



\## Key Findings

\- High monthly charges increased churn risk

\- Customers with short tenure were more likely to churn

\- Long-term contracts reduced churn
